Connect your Realtime Database so every Save & Publish syncs to the cloud — updating desktop and mobile instantly.

Connected
URL must be https://your-project-default-rtdb.firebaseio.com
📋
Setup Guide
▼
  1. Go to console.firebase.google.com
  2. Add project → name it brand-rehab
  3. Build → Realtime Database → Create database → Test mode
  4. Project Settings (gear) → scroll down → click </> to register web app
  5. Copy apiKey and projectId from the config block shown
  6. Database URL is shown in the Realtime Database panel
  7. Paste all three here and click Save & Connect
